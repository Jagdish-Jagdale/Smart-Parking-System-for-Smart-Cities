Project Purpose

The Smart Parking System aims to solve urban parking availability issues by demonstrating how IoT sensors can interface with dynamic web applications. Real-time updates help reduce driver frustration, carbon emissions, and parking congestion.

By utilizing simulated sensor events, this dashboard provides a visualization of how smart cities leverage REST APIs to monitor infrastructure state and display instant metrics on roadside signs.

Technology Stack

Python Flask
A micro-framework powering the REST API and state management.
Vanilla JS
Handles page routing, canvas rendering, dynamic grid building, and API polling.
Vanilla CSS
Responsive layouts, custom grid structures, and smooth interactive animations.
IoT Simulation
Mock server-side events and client actions that simulate physical sensors.
